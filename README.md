### About This Repository

This repository contains one of several small React portfolio introduction pages that I build as experiments and learning projects.

My **main portfolio website** lives in the `reyes2026/` repository. This repo acts more like a **journal of my intro sites**, where I try new layouts, animations, and UI ideas while continuing to improve my front-end development skills.

Each version reflects a different stage of my design and development process.
